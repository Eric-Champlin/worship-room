## Cross-Page Consistency Audit

### Inconsistencies Found

| Property | Page A | Page A Value | Page B | Page B Value | Severity |
|----------|--------|-------------|--------|-------------|----------|
| Hero gradient end color | Homepage | `#EDE9FE` (lavender) | Inner pages | `#F5F5F5` (neutral-bg) | INTENTIONAL -- Homepage has softer lavender transition to Journey section; inner pages transition to neutral bg |
| Hero radial extent | Homepage | `100% 60%` | Inner pages | `100% 80%` | INTENTIONAL -- Inner pages have wider radial spread |
| Hero gradient stops | Homepage | 35%/52% | Inner pages | 30%/55% | INTENTIONAL -- Slight variation for visual effect |
| Nav position | All pages | `absolute` | All pages | `absolute` | CONSISTENT |
| Prayer Wall hero | Prayer Wall | Uses `PageHero` component | Other pages | Different hero implementation | INTENTIONAL -- Prayer Wall has unique hero with HeadingDivider |
| Song Pick section | Daily Hub | Dark bg section | Homepage | Not present | INTENTIONAL -- Song Pick is Daily Hub only |
| Footer | All built pages | Present | Stub pages | Present | CONSISTENT |
| Squiggle viewBox | Pray/Journal tabs | `0 0 1800 1350` | Quiz section | `0 0 1200 1350` | INTENTIONAL -- Quiz squiggle is narrower variant |

### Observations

- All hero sections use the same gradient family (dark purple -> mid purple -> end color). The only difference is the end color (lavender on homepage, neutral-bg on inner pages).
- Navigation is always absolute-positioned over the hero gradient. There is no fixed/sticky nav variant.
- The glassmorphic pill style is mentioned in CLAUDE.md/memory but the nav container currently has transparent background with no backdrop-filter on either the landing page or inner pages. The pill effect comes from the overall hero gradient behind it.
- Footer is consistent across all pages -- same dark `#0D0620` background, same 3-column layout.
- Prayer Wall cards have no box-shadow (unlike meditation cards which have `shadow-sm`). Both use the same border treatment.
- All "Coming Soon" stub pages share the same layout: centered flex container with Inter H1 + Caveat "Coming Soon" text in primary color.
