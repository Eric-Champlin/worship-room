### Purple Glow Orb Gradients (LIVE values)

**CRITICAL — there are TWO glow systems in the app:**

**System 1: `GlowBackground` component (data-testid="glow-orb")**

Single-stop radial: `radial-gradient(circle, rgba(139, 92, 246, X) 0%, transparent 70%)` where X is the opacity.

| Variant | Orb count | Default opacity (from GlowBackground.tsx) | Color | Size (desktop) | Position | Live-rendered opacity on pages |
|---------|-----------|-------------------------------------------|-------|----------------|----------|-------------------------------|
| `center` | 1 | **0.15** | rgba(139, 92, 246) | 600×600px | top-30%, left-50% | **Overridden to 0.30** on Daily Hub tabs (Devotional, Pray, Journal) |
| `left` | 1 | **0.12** | rgba(139, 92, 246) | 500×500px | top-40%, left-20% | **Overridden to 0.30** on SongPickSection |
| `right` | 1 | **0.12** | rgba(139, 92, 246) | 500×500px | top-40%, left-80% | (not yet used in app) |
| `split` | 2 | **0.14** / **0.08** | rgba(139, 92, 246) + rgba(168, 130, 255) | 500×500 + 400×400 | left-25% + left-75% | **Overridden to 0.30** on MeditateTabContent |
| Mobile sizes | -- | -- | -- | 300×300 (center/left/right), 300×300+250×250 (split) | -- | Same opacity |
| Blur filter | -- | -- | -- | `blur(60px)` mobile / `blur(80px)` desktop | -- | -- |

**Every call site that uses `GlowBackground` in production today passes `glowOpacity={0.30}`.** The component's internal defaults (0.08-0.15) are NEVER rendered live — they are test fixtures.

### Observations

- **`bg-hero-bg` (#08051A) is the true app background.** It is set on `html`, `body`, and `#root` (per CLAUDE.md) to prevent white flash during transitions.
- **Two glow systems coexist:** `GlowBackground` (single-stop, 0.30 live) used on Daily Hub tabs + SongPickSection; inline two-stop/three-stop gradients on home (JourneySection, StatsBar, DashboardPreview, DifferentiatorSection, Quiz).
- **GlowBackground's internal defaults are effectively dead code.** Every production caller overrides `glowOpacity={0.30}`. The stated "0.25-0.50" range in CLAUDE.md is correct for homepage inline glows.
- **PageHero gradient is DIFFERENT from GlowBackground.** PageHero uses a subtle ellipse-at-top-center purple wash at `rgba(109, 40, 217, 0.15)` — same `#6D28D9` color the CTA uses. This is a softer, flatter treatment than the blur-80px orbs.
- **No page uses Caveat for primary headings anymore.** All headings have migrated to Inter + `GRADIENT_TEXT_STYLE`.
- **Section dividers appear only on the Home page and as interior dividers in Devotional tab.** Other pages rely on hero-pb → content padding for rhythm.
- **All FrostedCards have `backdrop-blur-sm` (4px).** Dashboard-style cards use `backdrop-blur-sm` too. Stronger blurs (`backdrop-blur-md`/`backdrop-blur-lg`) appear on sticky tab bar (16px) and specialized overlays.
