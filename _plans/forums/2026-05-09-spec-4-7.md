### Canonical modal pattern: `SleepTimerPanel` (Bible wave BB-28)

`frontend/src/components/bible/SleepTimerPanel.tsx` is the canonical rolls-own modal pattern in this codebase. ComposerChooser MUST follow it verbatim:

```tsx
import { createPortal } from 'react-dom'
import { useFocusTrap } from '@/hooks/useFocusTrap'

const focusTrapRef = useFocusTrap(isOpen, onClose)

return createPortal(
  <div className="fixed inset-0 z-50 flex items-center justify-center px-4">
    <div className="absolute inset-0 bg-black/40" aria-hidden="true" onClick={onClose} />
    <div ref={focusTrapRef} role="dialog" aria-modal="true" aria-labelledby="..." className="relative ...">
      ...
    </div>
  </div>,
  document.body,
)
```

`useFocusTrap(isActive, onEscape)` (`frontend/src/hooks/useFocusTrap.ts`) returns a ref. On `isActive=true` it captures `document.activeElement`, focuses the first focusable element inside the container, traps Tab/Shift+Tab cycling, listens for Escape (calling `onEscape`), and on cleanup restores focus to `previouslyFocused`. **This is the canonical primitive used by 37+ modals; do NOT roll a custom focus trap (W5/W21).**
